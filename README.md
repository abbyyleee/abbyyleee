## Featured Projects

### Project SOGAS — Southern Gas Services Website & Admin Dashboard

A full-stack application for a natural gas company in Louisiana, built as my first solo, end-to-end client project.

- Public marketing site built with **React + Vite** and **TailwindCSS** (custom theme)
- **Secure admin dashboard** for managing site content, gallery images, and customer inquiries
- **Node.js + Express** backend with validation, CORS, rate limiting, and health checks
- **Supabase (Postgres)** for persistent storage of content, gallery metadata, and contact submissions
- **Resend + SMTP (IONOS)** for contact form email notifications
- Environment-based configuration with `.env` and a clear separation between frontend, backend, and database

**Repo:**  
`https://github.com/abbyyleee/SOGAS` 

**Live site:**  
`https://www.sogasservices.com/`
